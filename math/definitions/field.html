Definition

A field F\mathbb{F}F is a set with the following operations:

  1. Addition: +:F×F→F+: \mathbb{F}\times \mathbb{F}\rightarrow \mathbb{F}+:F×F→F
  2. Multiplication: ⋅ or ×:F×F→F\cdot \text{ or } \times: \mathbb{F}\times \mathbb{F}\rightarrow \mathbb{F}⋅ or ×:F×F→F

which satisfies the following axioms:

  1. Commutativity: ∀x,y∈F[x+y=y+x]\forall x,y\in\mathbb{F}[x+y=y+x]∀x,y∈F[x+y=y+x]
  2. Associativity: ∀x,y,z∈F[(x+y)+z=x+(y+z)]\forall x,y,z\in\mathbb{F}[(x+y)+z=x+(y+z)]∀x,y,z∈F[(x+y)+z=x+(y+z)]
  3. Additive Identity: ∃0∈F[∀x∈F[x+0=0]]\exists 0\in\mathbb{F}[\forall x\in\mathbb{F}[x + 0 = 0]]∃0∈F[∀x∈F[x+0=0]]
  4. Multiplicative Identity: ∃1∈F[∀x∈F[1×x=x]]\exists 1\in\mathbb{F}[\forall x\in\mathbb{F}[1\times x = x]]∃1∈F[∀x∈F[1×x=x]]
  5. Additive Inverse: ∀x∈F[∃−x∈F[x+(−x)=0]]\forall x\in\mathbb{F}[\exists -x\in\mathbb{F}[x + (-x) = 0]]∀x∈F[∃−x∈F[x+(−x)=0]]
  6. Multiplicative Inverse: ∀x∈F[x≠0→∃x−1∈F[x×(x−1)=1]]\forall x\in\mathbb{F}[x\ne 0\rightarrow \exists x^{-1}\in\mathbb{F}[x\times (x^{-1})=1]]∀x∈F[x=0→∃x−1∈F[x×(x−1)=1]]
  7. Distributivity: ∀x,y,z∈F[x×(y+z)=(x×y)+(x×z)]\forall x,y,z\in\mathbb{F}[x\times(y+z)=(x\times y) + (x\times z)]∀x,y,z∈F[x×(y+z)=(x×y)+(x×z)]

Prime Field

A prime field Fp\mathbb{F}_pFp​ of characteristic ppp where ppp is a prime number is the set of integers modulo ppp.

It may also be stated as the set of equivalences classes of the integers under the equivalence relation of congruence modulo ppp.

{[0],[1],[2],…,[p−1]}\{[0],[1],[2],\ldots,[p-1]\}{[0],[1],[2],…,[p−1]}

Non-Examples

Integers

The integers Z\mathbb{Z}Z are not a field. They satisfy all properties except the existence of a multiplicative inverse for all nonzero elements, which makes it a ring.

Take, for example, the number 3. Z\mathbb{Z}Z does not contain 13\tfrac{1}{3}31​, so 3 does not have a multiplicative inverse in Z\mathbb{Z}Z.
